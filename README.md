# 🚦 Semáforo

Proyecto realizado con HTML, CSS y JavaScript puro.

## Funcionalidades

- Encender la luz roja.
- Encender la luz amarilla.
- Encender la luz verde.
- Apagar las demás luces al seleccionar una nueva.
- Cambio dinámico de estados mediante JavaScript.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

## Lo que aprendí

Durante este proyecto practiqué:

- Manipulación del DOM.
- Variables.
- Funciones.
- Eventos (`addEventListener`).
- Uso de clases CSS con JavaScript.
- Selección y modificación de elementos del DOM.
